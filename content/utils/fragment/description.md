# Parameters

- `...children`

  A list of child nodes. A child can either be a string or an HTML element (e.g. one generated by el()).

# Examples

```
fragment([
  // Using the el() snap.js function in this example.
  el('span', {}, ['Hello']),
  'World!',
])

/* The above results in the following HTML structure
<span>Hello</span>
World!
*/
```
